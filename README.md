# projet-bataille-navale
Projet bataille navale I1 S7 Thomas Debeaupte

La Bataille Navale est un jeu solo en Python où tu dois couler tous les bateaux placés aléatoirement sur une grille.
Le but : toucher tous les bateaux en un minimum de coups !

Lancer le jeu en executant : main.py
Puis : 
	•	Entre tes tirs avec une lettre pour la ligne et un chiffre pour la colonne, par exemple : B7
	•	Quitter le jeu à tout moment : tape Q ou QUIT

Affichage : 
 ~ : eau
 x : touché (affiché en rouge)
 o : raté (affiché en bleu)
